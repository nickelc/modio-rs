Sorting

See the modio docs for more information.

By default this returns up to 100 items. You can limit the result by using limit and offset.

Example

use modio::filter::prelude::*;
use modio::mods::filters::events::EventType as Filter;
use modio::mods::EventType;

let filter = Id::gt(1024).and(Filter::eq(EventType::ModfileChanged));
